Commands

Considering there's a specified prefix before each command

- help: Shows this command list
- ping: Ping the server specified in the config.json, returning the results
- start: Start the bot
- status: Status of the bot
- follow [username]: Follows the player with the exact username
- say [message]: Says [message] in chat. Disabled if "send-chat-to-minecraft" is set to true
- fish: Start fishing and stop when inventory is full
- stopfish: Stop fishing
- goto [x] [y] [z]: Go to [x] [y] [z] coordinates
- list: List inventory
- rawinv: List raw inventory in JSON
- empty: Empty inventory
- generator [x] [y] [z]: Breaks the block at [x] [y] [z] continuously
- stopgen: Stops the breaking process of the Generator command
- viewer: Returns the port and the URL in which the current world viewer is running
- clearlogs: Clears the logs folder
- stop: Stops any kind of pathfinding
- leave: Makes the Bot leave the server, Discord Bot will stay on
- exit: Stops the program
